Dependencies:

    Python3

Steps:

    >python3 DTerminal.py
        For Server(Executes commands):
            Press s/S. The server will start listening for connections
        For Client(Sends commands to be executed)
            Press c/C. The client sends the command in the following form:
            IP1>cmd1||IP2>cmd2...

            For eg.
            172.18.16.64>mkdir hola||172.18.16.31>date||172.18.16.31>touch t1.txt||172.18.16.31>ls
